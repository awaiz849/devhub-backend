DevHub Backend

A backend API for DevHub, a developer platform where users can register, log in, and interact with posts.
Built with Node.js, Express, and MongoDB as part of a learning project focused on backend development.

Tech Stack

Node.js

Express.js

MongoDB

Mongoose

bcryptjs (password hashing)

jsonwebtoken (JWT authentication)

Postman (API testing)

Features Implemented
Authentication

User registration

Password hashing using bcrypt

User login

JWT token generation for authentication

Backend Setup

Express server setup

MongoDB connection using Mongoose

Environment variables with dotenv

REST API structure
